# Hello World Skill

This is a simple example skill demonstrating the basic structure and functionality of Claude Code skills.

## Instructions

When this skill is triggered:

1. **Greet the user** - Provide a friendly, professional greeting
2. **Confirm skill activation** - Let the user know the skill is working correctly
3. **Provide helpful context** - Explain what this example skill demonstrates

## Example Responses

**User says:** "Hello world"

**Response:**
```
Hello! The hello-world skill is working correctly.

This example skill demonstrates:
- Basic SKILL.md structure with YAML frontmatter
- Clear instructions and examples
- Proper skill activation and response patterns

You can customize this skill or use it as a template for creating your own skills.
```

## Guidelines

- Keep responses friendly and professional
- Confirm that the skill is functioning properly
- Provide useful information about skill structure when asked
- This skill is primarily for testing and demonstration purposes
